# Ollama Terminal App

## Installation

1. Install Ollama first:
https://ollama.com

2. Download the latest release:
OllamaTerminalSetup.exe - releases page

3. Run installer

4. Open the app

5. Use qwen2:0.5b model

6. If you want better models, please run in command prompt - ollama pull (your model)


## Requirements
- Windows 10/11
- Ollama installed
