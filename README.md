# Ollama Terminal App

## Installation

1. Install Ollama first:
https://ollama.com

2. Download the latest release:
OllamaTerminalSetup.exe - releases page

3. Run installer

4. Open the app

5. Use qwen2:0.5b model

6. If you want better models, please run in command prompt - ollama pull (your model)

## File Storage

The app saves conversation memory and exported files to:
`C:\Program Files (x86)\OllamaTerminal\Memory\`

**Note:** This directory requires administrator privileges to create/write. If you encounter permission errors, the app will automatically fall back to your user directory: `C:\Users\[YourUsername]\OllamaTerminal\`

**Installation:** The installer automatically creates the Memory folder in the installation directory.

## Requirements
- Windows 10/11
- Ollama installed
